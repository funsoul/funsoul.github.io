![](/images/一段诡异的文本.png)

```
上图，红色文本为诡异文段，绿色文本为正常文档。无论怎么做（格式刷、改变字体、修改段间距等）都无法将红色文段的样式改为绿色文档的。

如果将其复制到记事本，另存为时，会出现下面的提示，因此可以判断，红色文段含有“特殊字符”。

```

![](/images/一段诡异的文本二.png)

```
使用notepad++打开，在红色文段使用“人肉搜索法”可以发现，确实是这样的。将其删除，并改为正常标点即可。
```
